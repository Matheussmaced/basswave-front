### 🌟 Visão Geral

O **Basswave Presets Hub** é uma plataforma moderna e intuitiva dedicada a baixistas, onde podem explorar e adquirir presets profissionais para diversos equipamentos de áudio (Ampero, Valeton, Boss, entre outros). Criado pelo experiente baixista e produtor **Luccas Macedo**, o projeto foca em entregar timbres de alta fidelidade prontos para o palco e estúdio.

🚀 **Status:** Em Produção

### ✨ Funcionalidades Principais

- 🌌 **Interface Dark Mode Premium**: Design focado na experiência do músico, com estética moderna.
- 🎨 **Efeitos Dinâmicos**: Fundo de partículas interativo e animações suaves.
- 📑 **Navegação de Presets**: Visualização detalhada de cada pack através de modais.
- 📺 **Integração de Vídeo**: Demonstrações reais via YouTube para cada preset.
- 📱 **Totalmente Responsivo**: Experiência otimizada para Desktop, Tablet e Mobile.

---

### 🛠️ Tecnologias Utilizadas

- **Core**: [Next.js 16](https://nextjs.org/) (App Router)
- **UI/Layout**: [Tailwind CSS 4.0](https://tailwindcss.com/)
- **Componentes**: [Radix UI](https://www.radix-ui.com/) (Dialog/Modal)
- **Ícones**: [Lucide React](https://lucide.dev/)
- **Animações**: [Framer Motion](https://www.framer.com/motion/) & Tailwind Animate
- **Tipografia**: Outfit & Inter (Google Fonts)

### 🚀 Como Executar o Projeto

#### Pré-requisitos
- [Node.js](https://nodejs.org/) (versão 18 ou superior)
- [npm](https://www.npmjs.com/) ou [yarn](https://yarnpkg.com/)

#### Passo a Passo

1. **Clone o repositório:**
   ```bash
   git clone <link-do-repositorio>
   cd basswave-front
   ```

2. **Instale as dependências:**
   ```bash
   npm install
   ```

3. **Inicie o servidor de desenvolvimento:**
   ```bash
   npm run dev
   ```

4. **Acesse no navegador:**
   Abra [http://localhost:3000](http://localhost:3000) para ver o resultado.
